# MetLife Performance Hub - Dashboard

Este repositório contém o painel interativo de análise de performance técnica e simulações de atendimento para o time da MetLife, desenvolvido e mantido pela rhapsody Intelligence.

## 🚀 Como Visualizar Localmente

Basta abrir o arquivo `index.html` em qualquer navegador da sua escolha! 

## 🌐 Como Publicar no GitHub Pages (Grátis)

Você pode publicar este painel diretamente na web usando o **GitHub Pages**:

1. Crie um novo repositório no seu GitHub (exemplo: `metlife-dashboard`).
2. Faça o upload dos seguintes arquivos desta pasta para o repositório:
   - `index.html`
   - `metlife_users_data.js`
   - `metlife_simulations_data.js`
   - `rhapsody_logo.png`
   - `README.md`
3. No seu repositório do GitHub, vá em **Settings** (Configurações) > **Pages** (no menu lateral esquerdo).
4. Em **Build and deployment**, selecione a branch `main` (ou `master`) e a pasta `/ (root)`.
5. Clique em **Save**.
6. Em alguns minutos, o GitHub fornecerá o link público para acessar o painel online! (exemplo: `https://seu-usuario.github.io/metlife-dashboard/`)
